# Jirah Visual Modes — Mode A and Mode B

Jirah produces two distinct deliverable aesthetics. **Pick the mode before you start designing.** Wrong mode undermines the deliverable.

---

## Mode A — JIRAH-Branded (Premium / Authority Voice)

**Used for:** Discovery docs, Discussion Documents, strategic framing, retention / succession proposals, pitches where JIRAH is the expert voice.

**Colors:**
- Gold: `#C5A55A`
- Navy: `#111827`
- Warm off-white background

**Fonts:**
- Playfair Display (serif display, headlines)
- DM Sans (body)

**Logo:** "J" monogram in gold.

**Feel:** Editorial, premium, slower, authoritative. Executive reading.

**Title treatment:** Use full formal branding — **"JIRAH Growth Partners"** — in the header/title.

**Exemplar files** (live in OneDrive — `[ONEDRIVE_ROOT]\04 - Deliverable Templates\Exemplars\`):
- `Discussion Document - S&A.pdf`
- `SA_Discussion_Document.html` — full-viewport slide-deck style with scroll-snap, fixed right-side nav dots, top progress bar, bottom-right slide counter

---

## Mode B — Co-Branded (Client as Home Team)

**Used for:** AI pilot plans, technical scoping docs, implementation proposals, work-product docs where the client is "home team."

**Colors:** Client's brand palette. E.g., Smith + Andersen uses navy `#003167` + orange `#C26A2B` + mustard `#FFB500`.

**Fonts:** Client's brand fonts. E.g., S+A uses Montserrat + Proxima Nova Condensed.

**Background:** White, charcoal ink (default — adjust to client's norm).

**Feel:** Work-product, technical, demo-first. The deliverable looks like it *belongs to* the client's firm, not Jirah's.

**Title treatment:** "Prepared by Jirah" or similar — lowercase "Jirah" in body copy. Jirah branding is small and secondary; client brand is primary.

**Exemplar file** (lives in OneDrive — `[ONEDRIVE_ROOT]\04 - Deliverable Templates\Exemplars\`):
- `Shop Drawing Pilot Plan.html` — Smith + Andersen AI pilot

## How to Choose

| Scenario | Mode |
|---|---|
| Discovery / Discussion Document | **A** |
| Strategic proposal (ops / succession / governance) | **A** |
| Pitch deck for new prospect | **A** |
| AI pilot plan | **B** |
| Technical scoping doc | **B** |
| Implementation proposal (ops engagement delivery plan) | **B** |
| Work-product artifacts shared with client team | **B** |
| LinkedIn post / public content | **A** (simplified) |

**Rule of thumb:** Is Jirah the narrator, or is the client the protagonist? Jirah narrates → Mode A. Client is the story → Mode B.
